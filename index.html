Política de Privacidad

Última actualización: 27/02/2026

Esta aplicación no recopila datos personales de los usuarios.

Uso de Internet

La aplicación utiliza conexión a Internet exclusivamente para descargar archivos CSV con información actualizada necesaria para su funcionamiento.

Enlaces externos

La aplicación puede redirigir a servicios externos como llamadas telefónicas, GoogleMaps o WhatsApp. Estos servicios se rigen por sus propias políticas de privacidad.

Datos recopilados

La aplicación no recopila, almacena ni comparte datos personales.
